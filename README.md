# Rover_NASA_Language
A simple program build as a laguage interpreter. The program must be able to authorize the user to manipulate by a command line a NASA Rover, moving left, right, up and down by a 2D structure.

Definir limite máximo e mínimo para o quanto o robô pode andar 
O tamanho da matriz deve ser 4 vezes o tanto que o robo pode andar
Tem que ter obstáculos na matriz
O robo não pode andar na parede
O robo gira no próprio eixo (E para esquerda, D para direita, G para 180 graus)
O robo só anda pra frente (A)
Antes de cada comando deve ter o número de indíce de quantas vezes irá ocorrer
